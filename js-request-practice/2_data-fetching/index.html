데이터 가져오기 (fetch GET)

fetch()는 브라우저에 내장된 HTTP 요청 함수입니다. 별도 설치 없이 사용할 수 있고, Promise를 반환합니다.
이 실습에서는 JSONPlaceholder API를 사용하여 사용자, 게시물, 댓글 데이터를 가져오는 연습을 합니다.

📦 사용할 API: JSONPlaceholder

무료 테스트용 REST API입니다. 실습에서 아래 엔드포인트를 사용합니다.

GET /users → 사용자 목록 (10명)
GET /users/:id → 특정 사용자 1명
GET /posts → 게시물 목록 (100개)
GET /posts?userId=N → 특정 사용자의 게시물
GET /comments?postId=N → 특정 게시물의 댓글

기본 URL: https://jsonplaceholder.typicode.com

섹션 1: fetch 기본 — then/catch

fetch()는 Promise를 반환합니다. .then()으로 응답을 받고, .json()으로 데이터를 파싱합니다.

💡 fetch()의 반환값은 Response 객체입니다. 데이터를 사용하려면 response.json()으로 한 번 더 파싱해야 해요.

섹션 2: Response 객체 살펴보기

fetch가 반환하는 Response 객체에는 상태 코드, 헤더 등 유용한 정보가 담겨 있습니다.

⚠️ response.ok는 상태 코드가 200~299일 때 true입니다. 404 등의 에러에서도 fetch 자체는 실패하지 않으므로, 반드시 response.ok를 확인해야 합니다!
