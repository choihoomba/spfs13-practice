JSON 다루기 연습

JSON(JavaScript Object Notation)은 데이터를 주고받을 때 사용하는 텍스트 형식입니다.
이 실습에서는 JSON.stringify(), JSON.parse()를 사용해 데이터를 변환하고 처리하는 방법을 연습합니다.

섹션 1: JSON.stringify() — 객체를 JSON 문자열로

자바스크립트 객체를 JSON 문자열로 변환합니다. HTTP 통신에서는 문자열만 전송할 수 있기 때문에, 서버로 데이터를 보낼 때 반드시 문자열화가 필요합니다.

💡 왜 문자열로 바꿔야 할까?
HTTP 프로토콜은 '문자열'만 통과할 수 있어요. 자바스크립트 객체를 직접 보낼 수 없기 때문에 JSON.stringify()로 문자열화를 시켜야 합니다.

원본 객체 (typeof: object)

여기에 원본 객체가 표시됩니다.

JSON 문자열 (typeof: string)

보기 좋게 포맷팅된 JSON이 여기에 표시됩니다.
