유틸리티 모듈 만들기 실습

export default와 named export를 활용하여 재사용 가능한 유틸리티 모듈을 만드는 실습입니다.
각 섹션의 안내를 따라 js/ 폴더 안의 파일들을 완성하세요.

파일 구조

이번 실습에서 작업할 파일들입니다.

2_utility-module/ ├── index.html ← 지금 보고 있는 파일 ├── js/ │ ├── main.js ← 메인 실행 파일 (import 작성) │ ├── greet.js ← 인사 함수 (default export) │ ├── config.js ← 설정 객체 (default export) │ ├── math-utils.js ← 수학 유틸 함수들 (named export) │ └── string-utils.js ← 문자열 유틸 (default + named 혼합)
index.html은 이미 <script type="module" src="js/main.js">로 연결되어 있습니다.
여러분은 js/ 폴더 안의 5개 파일만 수정하면 됩니다.

참고: named export vs default export

구분 named export default export
개수 여러 개 가능 파일당 1개만
내보내기 export function add() {} export default function add() {}
가져오기 import { add } from '...' import add from '...'
중괄호 필요 { } 필요 없음
이름 변경 as 키워드 필요 자유롭게 가능
비유 메뉴판에서 골라 주문 대표 메뉴 주세요
외우는 팁: default = "대표" = 하나니까 그냥 쓰기, named = "이름표" = 여러 개니까 { }로 골라쓰기

섹션 1: default export — greet.js

1. js/greet.js를 열고 함수에 export default를 붙이세요.

2. js/main.js에서 greet.js의 함수를 import 하세요.

export default — 파일의 "대표 선수"를 지정합니다. 파일당 1개만 가능해요.
예: export default function greet(name) { ... }

import — default export는 중괄호 없이 가져옵니다.
예: import greet from './greet.js';
주의! default export를 가져올 때 { }를 쓰면 에러가 납니다!
import { greet } from './greet.js'; ← 이러면 안 돼요!
테스트 버튼을 눌러 결과를 확인하세요.

섹션 2: 객체 default export — config.js

1. js/config.js를 열고 설정 객체에 export default를 붙이세요.

2. js/main.js에서 config.js의 객체를 import 하세요.

함수뿐 아니라 객체도 default export 할 수 있어요!
예: export default { siteName: '블로그', theme: 'dark' };

default export는 import 할 때 이름을 자유롭게 정할 수 있어요.
예: import siteConfig from './config.js'; — 원래 이름과 달라도 OK!
테스트 버튼을 눌러 결과를 확인하세요.

섹션 3: named export — math-utils.js

1. js/math-utils.js를 열고 함수와 상수에 export를 붙이세요.

2. js/main.js에서 필요한 함수와 상수를 import 하세요.

파일에 기능이 여러 개일 때는 named export가 편리해요.
예: export function add(a, b) { ... }
가져올 때: import { add, subtract } from './math-utils.js';

named export는 이름이 고정이라 중괄호 { } 안에 정확한 이름을 써야 해요.
테스트 버튼을 눌러 결과를 확인하세요.

섹션 4: default + named 혼합 — string-utils.js

1. js/string-utils.js를 열고 대표 함수에 export default, 나머지에 export를 붙이세요.

2. js/main.js에서 default와 named를 한 줄에 가져오세요.

한 파일에서 default 1개 + named 여러 개를 동시에 내보낼 수 있어요!
가져올 때: import formatName, { toUpperCase, addExclamation } from './string-utils.js';

순서 주의! default가 항상 앞에, named가 뒤에 와야 해요.
import formatName, { toUpperCase } from '...'; — 올바른 순서!
import { toUpperCase }, formatName from '...'; — 에러!
테스트 버튼을 눌러 결과를 확인하세요.
