모듈 분리하기 실습

하나의 파일에 모든 코드가 들어있는 상태에서, 기능별로 파일을 나누고 export / import를 사용해 연결하는 실습입니다.
각 섹션의 안내를 따라 js/ 폴더 안의 파일들을 완성하세요.

파일 구조

이번 실습에서 작업할 파일들입니다.

1_module-seperation/ ├── index.html ← 지금 보고 있는 파일 ├── js/ │ ├── main.js ← 메인 실행 파일 (import 작성) │ ├── utils/ │ │ └── math.js ← 계산 함수 (export 작성) │ ├── data/ │ │ └── products.js ← 상품 데이터 (export 작성) │ └── ui/ │ └── render.js ← 화면 표시 함수 (export 작성)
index.html은 이미 <script type="module" src="js/main.js">로 연결되어 있습니다.
여러분은 js/ 폴더 안의 4개 파일만 수정하면 됩니다.

섹션 1: export + import — utils/math.js

두 파일을 수정합니다:

1. js/utils/math.js를 열고 함수에 export를 붙이세요.

2. js/main.js를 열고 math.js에서 함수를 import 하세요.

export — 함수 앞에 export를 붙이면 다른 파일에서 가져다 쓸 수 있어요.
예: export function add(a, b) { return a + b; }

import — import { 함수명 } from './경로/파일.js'; 형태로 가져와요.
예: import { add } from './utils/math.js';
